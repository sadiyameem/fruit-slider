# Fruit Slider

# Description
- Fruit Slider is an interactive project that shows different fruits using a vertical slider.
- Each slide has a fruit image, name, and fun fact about it.

# Features
- Vertical sliding animation
- Up and down arrows
- Color-themed slides for each fruit
- Images of the fruit
- Fun fruit facts

# Tools
- HTML
- CSS
- JavaScript
